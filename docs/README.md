# Chatbot_CN 说明文档

Chatbot_CN是一个基于第二代和第三代机器人技术结合的对话机器人项目，目前该项目的主要目标
为任务型多轮对话，主要的目的为解决多场景、多目标、多任务的较为复杂的任务型机器人。主要宗旨为利用最先进的人工智能技术
解决目前行业内重复的任务型客服工作。
